오에에

윈도우 cmd 커맨드

  • cd : 위치이동
  • dir : 해당 폴더 안에 파일들 보기
  • mkdir : 새 폴더 만들기
  • code : 비쥬얼 스튜디오 여는 커맨드

단축키, 단축 명령어들

  • index.html에서

    • html + tap - html 기본 문서 양식을 들고온다
    • link + tap - link 하고싶은 파일을 들고온다
    • script + tap - script 태그를 자동 세팅

  • main.js에서

    • clg - console.log()를 가져온다

  • 기타

    • ctrl + p : 가고싶은 파일로 이동
    • ctrl + shift + p : 실행 가능한 명령어들을 볼 수 있다
    • ctrl + / : 문장 주석처리, 주석은 컴파일 시 건너뛴다.
    • ctrl + s : 문서 저장
